BepInEx patchers list:
https://github.com/BepInEx/BepInEx.Debug

How to install patchers:
- as a mod: put the `.dll` into 
	`your-mod\patchers\`
- in the RW folder:
	- drop into 
		`Rain World\BepInEx\patchers\`
	- add patcher name to 
		`Rain World\RainWorld_Data\StreamingAssets\whitelist.txt`
